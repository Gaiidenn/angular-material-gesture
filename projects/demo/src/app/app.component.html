Angular material tabs gesture

This demo uses @angular-material-gesture/mat-tab-group-gesture module to enable gesture behavior on mat-tab-group component

If you are on a MOBILE PLATFORM (smartphone / tablet / any device with touchscreen) you should now be able to :
  • Scroll tab header with your finger IF there are too much tabs to show it all on your screen
  • Swipe between tabs by swiping the tab content from left to right (or right to left)

PRO TIP for DESKTOP users :

If you are on a desktop platform, use your browser's developper tools to emulate some smartphone or tablet device and test this gesture features !

  • Google Chrome : Press F12, click the "toggle device bar" button in the top left corner, then select whatever touch device you want to try!
  • Quick guide for most used browsers : https://www.digitalcitizen.life/emulate-mobile-device-desktop-browser/
